2. Primeton BIIP 7.0简介：
    Primeton BIIP 是普元软件提供的基于SOA理念和SEDA架构，面向交易类软件的产品.
    
2.1 BIIP 7.x 简介：

    普元软件在2010年推出了Primeton BTP（以下简称BTP）产品，用于支撑交易类系统。 Primeton BIIP 由BTP产品发展而来，在BTP的技术架构之上改进了交易开发模式，完善了报文转换、交易流水等组件的不足，同时增加了对批量转联机、对账、复核等业务场景的支持，此外在主数据管理、部署管理、应用监控等方面也有所补充和完善。基于BIIP，可以支撑银行中间业务、综合前置等集成类业务系统建设。


    2.1.1   BIIP Server (BIIP 运行平台)
        运行平台基于JVM，为各功能组件提供运行环境。一笔交易从接入、报文解包、交易处理、报文打包，再到交易接出的整个运行过程都由运行平台进行统一的调度。

    2.1.2   BIIP Studio (BIIP 开发平台)
        BIIP Studio是集面向构件应用的设计、开发、组装、调试、维护、部署、管理和发布于一体的集成开发环境，提供对交易应用全生命周期的开发、维护和管理。在BIIP Studio中，以项目的形式组织了BIIP 应用开发的资源，提供相应的向导、视图和编辑器等工具供开发人员在开发过程中可视化地开发各种构件，并提供了强大的调试及团队开发功能。

    2.1.3    BIIP Governor（BIIP 管理门户）
        BIIP Governor可以对运行平台进行构件包的部署，升级，系统参数的设置和日志级别设置，日志文件下载。

    2.1.4    BIIP Monitor (BIIP 监控门户)
        BIIP Monitor主要功能是以图形化的方式实现对BIIP系统运行时刻进行监控，以利于系统开发人员及运行管理人员进行系统调试与系统诊断。通过实时在线监控和管理工具，可以实现对应用系统各个层次进行监控和管理。用户只需通过Web界面即可实时监控应用系统的各项运行参数，快速诊断和修正系统运行时的错误及异常，用更少的维护成本确保系统正常发挥作用。

    2.1.5    BIIP Workspace（BIIP 业务门户）
        BIIP 业务门户对业务组件进行管理，可以进行冲正查询，对账查询、流水查询等等。
